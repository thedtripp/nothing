# nothing-site

Public deploy-only repo for the Nothing™ landing page. Served via GitHub Pages.

This repo contains **only** the static, publicly-shippable site — no strategy, no marketing brainstorm, no secrets. Source of truth for planning lives in the private `nothing` repo.

Rule: nothing with a secret value (Stripe keys, webhook signing secrets, etc.) ever gets committed here. Those go in the hosting provider's secret store when the backend exists.
